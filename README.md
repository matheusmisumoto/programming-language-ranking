# Web Programming Languages Ranking
Web programming languages ranking, based on webscraping from Google, GitHub and StackOverflow.
Created by [Felipe Fernandes](https://github.com/felipe507), [Matheus Misumoto](https://github.com/matheusmisumoto) and [Matheus Oliveira](https://github.com/MathOli), from the Internet Systems course of São Paulo State Technological College.

# Ranking de Linguagens de Programação
Script de criação de um ranking de linguagens de programação web, utilizando JavaScript e raspagem de dados do Google, GitHub e StackOverflow.
Criado por [Felipe Fernandes](https://github.com/felipe507), [Matheus Misumoto](https://github.com/matheusmisumoto) and [Matheus Oliveira](https://github.com/MathOli).
Apresentado como trabalho na disciplina de Programação de Sítios Internet do curso de Sistemas para Internet da Faculdade de Tecnologia do Estado de São Paulo (FATEC Baixada Santista - Rubens Lara).
